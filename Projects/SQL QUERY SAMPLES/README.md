# SQL Query Sample

## Overview
This project demonstrates SQL queries used for data analysis and reporting.

## Skills Demonstrated
-Sorting (ORDER BY)
-Filtering (WHERE)
-Aggregations (SUM, AVG, COUNT)
-Grouping (GROUP BY)
-Calculated Columns
-CASE Statements
-Window Functions (RANK)
-INNER JOIN
-LEFT JOIN
-HAVING
-Common Table Expressions (CTEs)
